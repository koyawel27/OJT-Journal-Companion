# OJT Journal Companion

OJT Journal Companion is a personal offline-first journal companion for students and interns who need to record daily OJT activities, track rendered hours, organize photo documentation, and prepare weekly journal content before transferring it into an official school journal template.

## Current Status

Planning stage.

## Intended Stack

- HTML
- CSS
- JavaScript
- IndexedDB
- Later PWA support

## Project Note

This is a personal offline-first journal companion for one student using the app locally. It is not a full internship management system and does not include login, admin dashboards, supervisor approval, cloud sync, GPS attendance, or multi-user features for v1.0.
